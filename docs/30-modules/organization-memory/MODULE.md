# Module: Organization Memory

## Purpose

Organization memory captures standards imported or adopted from a company, team, platform group, or
enterprise.

## Owns

- Organization standards concept.
- Future `adopt` and `import standards` direction.
- Relationship between organization standards and repository decisions.

## Does Not Own

- Runtime import implementation in P1.5.
- External MCP integrations.
- Enterprise policy enforcement.

## Current Decision

Organization memory is a future layer, not an MVP runtime feature.
